# لبن العصفور — Laban Alasfour

## التعريف المختصر

لبن العصفور هو مسار التوريدات العامة والخامات والمنتجات النادرة أو صعبة الوصول، مع توجه مستقبلي لعرض المنتجات وتجربتها افتراضيًا.

## التعريف التفصيلي

Laban Alasfour يخدم احتياجًا عمليًا متكررًا في التنفيذ والصيانة: الوصول إلى خامة، قطعة، بديل، أو منتج مناسب عندما لا يكون متاحًا بسهولة. المسار يربط الطلب بالصورة أو الكود أو المقاس أو الاستخدام، ثم يحدد هل المطلوب نفس المنتج، بديل مطابق، بديل اقتصادي، أو توريد مرتبط بمشروع.

## التموضع الاستراتيجي

مسار توريدات وخامات ومنتجات معمارية يصعب الوصول إليها، يخدم التنفيذ والصيانة والتشطيب، ويستعد مستقبلًا لتجارب AR/VR وملفات CAD/GLB.

## الخدمات

- توريد خامات ومستلزمات تنفيذ
- البحث عن منتجات وقطع نادرة أو بدائلها
- توريدات تخدم الصيانة والتشطيب والتجهيز التجاري
- تنظيم بيانات المنتجات والصور والمقاسات
- دعم ملفات CAD/GLB/AR/VR للمنتجات المستقبلية
- ربط التوريد باحتياجات المشاريع والفروع

## الجمهور المستهدف

- المهندسون والمقاولون وأصحاب المشروعات
- فرق الصيانة التي تحتاج قطع أو بدائل
- فرق المشتريات والتوريدات
- العملاء الذين يبحثون عن خامة محددة أو بديلها

## أسئلة جمع البيانات

1. ما نوع الخامة أو المنتج المطلوب؟
2. هل توجد صورة أو كود أو مقاس؟
3. ما الكمية المطلوبة؟
4. هل المطلوب بديل أم نفس المنتج؟
5. هل التوريد لمشروع، صيانة، أم تشطيب؟
6. ما موعد الاحتياج؟

## الرؤية المستقبلية

تعتمد الرؤية المستقبلية على تنظيم المنتجات بملفات **JSON وGLB وCAD** وصور متعددة لدعم تجارب **AR/VR** والتصميم الذكي، مما يتيح معاينة المنتجات افتراضيًا قبل الشراء أو التنفيذ.

## سياسة التحويل

يُحوَّل إلى Laban Alasfour عند وجود طلب خامة أو قطعة أو منتج أو بديل أو توريد مرتبط بمشروع أو صيانة أو تشطيب.
