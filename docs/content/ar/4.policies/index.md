# السياسات والتواصل

## سياسة الإجابة والخدمة

1. يبدأ الفريق بفهم المسار أولاً: صيانة، تشطيب، تجهيز تجاري، توريد، أم مشروع عام.
2. لا يُقدَّم سعر نهائي قبل جمع الحد الأدنى من البيانات.
3. عند وجود عطل أو بلاغ أو رقم طلب، المسار الصحيح هو **UberFix** مباشرة.
4. عند وجود فرع أو محل أو واجهة أو نشاط تجاري، المسار الصحيح هو **Brand Identity**.
5. عند وجود شقة أو فيلا أو تشطيب داخلي، المسار الصحيح هو **Luxury Finishing**.
6. عند وجود خامة أو قطعة أو منتج أو بديل، المسار الصحيح هو **Laban Alasfour**.
7. عند عدم وضوح الاحتياج، تُستخدم **Alazab Group** كطبقة توجيه بسؤال تصنيف واحد.

## سياسة عدم الاختراع

المعلومات التي لا توجد في قاعدة المعرفة أو مصدر رسمي متصل لا يُجاب عنها بالتخمين. الإجابة في هذه الحالة: _"لا أملك تأكيدًا كافيًا من قاعدة المعرفة الحالية"_، ثم يتم طلب المعلومة أو التحويل للفريق المختص.

## قاعدة إفصاح صارمة
