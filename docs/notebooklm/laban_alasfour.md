# لبن العصفور — Laban Alasfour

## 1. وظيفة هذا الملف
هذا الملف هو المرجع التشغيلي والمعرفي لمسار **لبن العصفور** داخل قناة معرفة مجموعة العزب. يستخدمه الموقع، البوت، Rasa، RAG، MCP، وفريق خدمة العملاء لتقديم ردود متسقة بدون اختراع أو تضارب.

## 2. التعريف المختصر
Laban Alasfour هو مسار التوريدات العامة والخامات والمنتجات النادرة أو صعبة الوصول داخل مجموعة العزب.

## 3. التعريف التفصيلي
لبن العصفور يخدم احتياجًا عمليًا متكررًا في التنفيذ والصيانة: الوصول إلى خامة، قطعة، بديل، أو منتج مناسب عندما لا يكون متاحًا بسهولة. المسار يربط الطلب بالصورة أو الكود أو المقاس أو الاستخدام، ثم يحدد هل المطلوب نفس المنتج، بديل مطابق، بديل اقتصادي، أو توريد مرتبط بمشروع. الرؤية المستقبلية تشمل عرض المنتجات بملفات منظمة مثل JSON وGLB وCAD وصور متعددة لتجارب AR/VR والتصميم الذكي.

## 4. التموضع الاستراتيجي
مسار توريدات وخامات ومنتجات معمارية يصعب الوصول إليها، يخدم التنفيذ والصيانة والتشطيب ويستعد مستقبلًا لتجارب AR/VR وCAD/GLB.

## 10. سياسة التحويل
حوّل إلى Laban Alasfour عند وجود طلب خامة أو قطعة أو منتج أو بديل أو توريد مرتبط بمشروع أو صيانة أو تشطيب.

## 11. أسلوب الصوت والمحادثة
عملية ومرنة؛ تركّز على فهم المنتج المطلوب بدقة، ثم البحث عن نفس المنتج أو بديله المناسب.

## 12. ما يجب فعله
- اطلب صورة أو كود أو مقاس عند الإمكان
- اسأل عن الكمية وموعد الاحتياج
- حدد هل المطلوب نفس المنتج أو بديل

## 13. ما يجب تجنبه
- لا تؤكد توافر منتج بدون تحقق
- لا تخلط بين طلب التوريد وطلب الصيانة
- لا تعطي سعرًا نهائيًا قبل تحديد المواصفات والكمية

## 14. افتتاحية مقترحة للمحادثة
> أهلاً بحضرتك، لمساعدتك في التوريد نحتاج صورة أو كود أو مقاس المنتج، والكمية المطلوبة، وهل المطلوب نفس المنتج أم بديل مناسب.

## 15. رسالة تحويل مقترحة
> تم تحديد الطلب كمسار توريد. سنحتاج مواصفات المنتج والكمية وموعد الاحتياج قبل أي تأكيد توافر أو سعر.
